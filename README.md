# Quiz Application Backend

A RESTful Quiz Application backend built with Spring Boot.

## Tech Stack
- Spring Boot
- Spring Data JPA
- MySQL
- Swagger (OpenAPI)
- Maven

## Features
- Create Quiz
- Random Question Generation
- Attempt Quiz
- Score Calculation
- Pagination
- DTO Layer
- Global Exception Handling
- Swagger API Documentation

## API Endpoints

Create Quiz

POST /quiz/create

Get Quiz Questions

GET /quiz/get/{id}

Submit Quiz

POST /quiz/submit/{id}

## Run the Project

1. Clone the repository
2. Configure MySQL database
3. Run Spring Boot application
